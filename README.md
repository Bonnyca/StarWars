## Installation
### 1. Clone the repo.
```console
$ git clone https://github.com/Bonnyca/StarWars.git
```
### 2. Change into the directory & install dependencies
```console
$ cd StarWars
$ npm install
```
### 3. Run the server:
```console
$ npm start
```

Runs the app. Navigate your browser to [http://localhost:3000](http://localhost:3000) to access the application.
